# Plan: Editor UX features (wzór Creatomate)

## Context
Funkcje UX edytora Creatomate, które poprawiają workflow montażysty.

## UX features w Creatomate

### Canvas / Designer
- Drag & drop elementów na canvas
- Resize handles
- Alignment tools (snap to grid, snap to elements)
- Multi-select i grupowanie

### Timeline
- Track-based layering
- Drag elementów na timeline
- Keyframe markers na timeline
- Zoom in/out na timeline

### Inspector / Properties
- Panel właściwości po prawej
- Keyframe "+" button przy każdej właściwości
- Animation picker
- Responsive preview (różne aspect ratio)

### Edytor kodu
- **JSON/Source editor** — pełna edycja szablonu jako JSON
- Przełączanie między visual editor a code editor
- Syntax highlighting

### Historia
- **Version history** — pełna historia zmian
- Undo/redo

### Preview
- **Real-time preview** — podgląd na żywo
- Play/pause/seek

### Responsive
- **Responsive templates** — jeden szablon, wiele formatów
- Automatyczna adaptacja do aspect ratio

### Fonts
- **Google Fonts** — tysiące fontów
- **Custom font upload** — WOF/OTF/TTF

## Co mamy w naszym edytorze
- Canvas z elementami ✅
- Timeline z trackami ✅
- Properties panel (inspector) ✅
- Playback (play/pause/seek) ✅
- Undo/redo (history) ✅
- Resize handles ✅
- Multi-select ✅

## Czego brakuje (priorytet)
1. **Alignment/snap tools** — snap to grid, snap to elements, alignment guides
2. **JSON/Source editor** — podgląd i edycja composition jako JSON
3. **Responsive preview** — podgląd w różnych aspect ratio (9:16, 1:1, 16:9)
4. **Google Fonts picker** — przeglądanie i wybór z Google Fonts
5. **Custom font upload** — upload własnych fontów
6. **Keyframe markers na timeline** — wizualizacja keyframe'ów
7. **Animation picker** — panel z gotowymi animacjami do wyboru
8. **Version history** — zapisane wersje (nie tylko undo/redo)
