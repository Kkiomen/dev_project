# Creatomate — przegląd platformy

## Czym jest Creatomate
Platforma do automatycznego tworzenia video i obrazów. Edytor webowy (template editor) + REST API + No-code integracje. Cały system oparty na szablonach, które można parametryzować danymi.

## Typy elementów
| Element | Opis |
|---------|------|
| Video | Klipy wideo z URL/upload, AI-generowane |
| Image | Obrazy statyczne |
| Audio | Klipy audio, AI text-to-speech |
| Text | Tekst z auto-sizing, tysiące fontów, emoji |
| Shape | Free-form shapes, prostokąty, koła, custom paths |
| Composition | Grupy elementów jako sceny (nested) |

## Eksport
- MP4 (video), JPG (obrazy), GIF (animowane)
- Rendering w chmurze, batch processing (100+ z CSV)

## Czego brakuje (z recenzji)
- Import After Effects templates
- Layer scripting / dynamic content sync
- Bezpośrednie integracje z Google Sheets/Drive/Dropbox
- Ograniczenia z loopującym audio/video
- Może być drogi przy dużej skali (credit-based pricing)
